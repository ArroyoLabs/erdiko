Download and Install Erdiko

  1. Clone Erdiko from github.

    git clone https://github.com/arroyo/Erdiko

  2. Set up an apache vhost to the webroot.
  3. Set the Webroot to /www/public/

Heads up! Theme/view files go in the /www/public folder while application code goes in the /www/app/ folder. Do not modify files outside of app and public folder if you want to maintain an easy upgrade path with Erdiko.

Create a new page

  1. Open Erdiko/www/app/config/contexts/default.json
  2. Add the following code to menu session 
    	,
            					{
            					"href":"/examples/bmi",
            					"title":"BMI"
            					}
  3. After added the above code, it should look like the following
    , { "href":"/examples/bmi", "title":"BMI" } ........ .......
  4. Open Erdiko/www/app/controllers/examples.php
  5. Add the following function after the last function. 
    	
    public function bmiAction()
	{
		$this->setTitles('BMI Example');
		//$this->setBodyContent(".....Test......");
		$this->setView('examples/bmi.php');
		$this->addJs('/themes/hello/js/example.js');
	}
  6. After added the above code, it should look like the following
    ........ .......
  7. Open your web browser to view the page
